## 概要
COACHTECH 教材 Tutorial 7-1「PHPの基礎 ハンズオン演習」で作成した成果物です。
（**商品価格計算プログラム**）

## 使用技術
- PHP 8.x
- HTML5

## 学んだこと
- 変数の定義
- 変数を用いた計算処理
- 変数を組み込んだHTML文書の作成
- 演算子を用いた判定プログラムの作成

## 動作確認
- 1.ターミナル上でphp-practiceディレクトリに移動（cd ~/php-practice/src/7-1-6_hands-on/practice）
- 2.Dockerデスクトップ版を立ち上げてから、ターミナル上でDocker環境を起動（docker compose up -d）
- 3.ブラウザのアドレスバーに「http://localhost:8000/7-1-6_hands-on/practice/price_calculator.php」を打ち込んで接続し、内容がちゃんと表示されているかを確認
